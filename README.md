# Miraim ログイン画面 UI開発

婚活男性向け内面スタイリングアプリ「Miraim」のログイン・新規登録画面のUI開発用リポジトリです。

## 📖 **プロジェクト背景**

### **アプリ概要**
結婚相談所に通う真剣な婚活男性のコミュニケーション能力を向上させ、理想的なパートナーとの出会いを実現するトータルサポートアプリです。

### **ターゲットユーザー**
- **結婚相談所入会者**: 多額の入会金を支払い、真剣に結婚を考えている男性
- **コミュニケーション課題**: 「口下手」「話がうまくいかない」と自覚、特に初対面への不安
- **精神的サポート必要**: 婚活疲れ、「落ち込む」「何をしていいか分からない」状態

### **4つの主要機能**（将来実装予定）
1. **AIカウンセラー**: 婚活悩み相談・自己紹介文作成支援
2. **会話練習機能**: 聞く力特化のコミュニケーショントレーニング
3. **相性診断機能**: 性格・価値観診断とマッチング支援
4. **スタイリング提案**: 年齢・季節・タイプ別の外見改善提案

### **今回の開発スコープ**
上記機能の入り口となる**ログイン・新規登録画面**のUI開発のみを対象とします。

## 🎯 開発目標

- **ログイン画面**: 既存ユーザーの認証
- **新規登録画面**: 新規ユーザーの登録
- **フレンドリーなUI**: 親しみやすく、温かみのあるデザイン
- **レスポンシブデザイン**: モバイル・デスクトップ対応
- **バリデーション**: フォーム入力検証
- **エラーハンドリング**: 適切なエラー表示

## 🎨 **デザインコンセプト**

### **UI/UXの方向性**
- **フレンドリーで親しみやすい**: 婚活への不安を和らげる温かいデザイン
- **安心感のある色合い**: 優しいブルーとグリーンを基調とした配色
- **シンプルで直感的**: 複雑さを避け、使いやすさを重視
- **励ましの要素**: ユーザーが前向きになれるような文言やアイコン

### **参考デザイン**
- `RFPimage/` フォルダ内の画像を参考にしてください
- 特に会話画面のフレンドリーな雰囲気を参考に
- 親しみやすさと信頼感のバランスを重視

## 🛠️ 技術スタック

- **フレームワーク**: Next.js 15.2.4
- **スタイリング**: Tailwind CSS 3.4.1
- **フォーム**: React Hook Form 7.50.0
- **UI コンポーネント**: Material-UI 5.15.7
- **HTTP クライアント**: Axios 1.6.7

## 🚀 開発環境セットアップ

### 1. 依存関係のインストール
```bash
npm install
```

### 2. 開発サーバー起動
```bash
npm run dev
```

### 3. アクセス
- フロントエンド: http://localhost:3000
- ログイン画面: http://localhost:3000/auth/login
- 新規登録画面: http://localhost:3000/auth/register

## 📋 API仕様

### 認証API（現在実装済み）
- **ベースURL**: `http://localhost:8000`
- **ログイン**: `POST /auth/login`
- **新規登録**: `POST /auth/register`

詳細は `docs/API_SPECIFICATION.md` を参照してください。

### 将来実装予定API
- **AIカウンセラー**: `/api/counselor/chat`
- **会話練習**: `/api/conversation/practice`
- **相性診断**: `/api/compatibility/diagnose`
- **スタイリング提案**: `/api/styling/recommend`

※現在の開発スコープは認証API のみです。

## 🎨 デザイン要件

### カラーパレット（既存miraim/frontendに準拠）

## 🎯 完成イメージ

### ログイン画面
- **メールアドレス・パスワード入力**
- **ログインボタン**（フレンドリーなデザイン）
- **新規登録リンク**
- **パスワード忘れリンク**

### 新規登録画面

#### **必須取得項目**
- **氏名**（ニックネーム可）
- **メールアドレス**
- **パスワード**
- **生年月日**
- **婚活状況**（選択式：結婚相談所利用中、マッチングアプリ利用中、その他）

#### **任意取得項目**
- **職業**
- **出身地**
- **居住場所**
- **趣味**
- **休日の過ごし方**

#### **その他要素**
- **利用規約・プライバシーポリシー同意**
- **新規登録ボタン**（温かみのあるデザイン）
- **ログインリンク**

### **フォーム設計のポイント**
- **段階的入力**: 必須項目と任意項目を分けて表示
- **進捗表示**: ステップ表示で完了感を演出
- **励ましメッセージ**: 「あと少しで完了です！」等の前向きな文言
- **バリデーション**: リアルタイムでの入力チェック
